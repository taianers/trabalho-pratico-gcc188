# Trabalho Pratico GCC188 - Receitas Buchinho Cheio

## Tecnologias Utilizadas
### Front-end:
* Vue.js 2.6.10
### Back-end:
* Node.js 16.2.0
### Banco de dados:
* MySQL 8.0.21
